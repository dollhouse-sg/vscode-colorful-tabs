## Installation

Search **Colorful Tabs** in the Extensions panel (`Ctrl+Shift+X`)

## Markers not showing?

| Setting | Value |
|---|---|
| `workbench.editor.decorations.badges` | `true` |
| `workbench.editor.tabSizing` | `fit` or `fixed` |
